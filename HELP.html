CT Seller Image Helper Chrome Extension

Jason Norris CTFTW 7/27/2025

Terms, Conditions & Privacy Notice is available in terms.html

Objective:

This extension will download all the cover photos, and optionally all item photos, for a specified CTBids auction from the Seller Dashboard, using the Lot title as the filename. This makes it easy to not only identify the photo to the lot, but also provide a meaningful description when creating email & social marketing campaigns, uploading photos to EstateSales.net/org/com, et al, etc.  

History:

My first version, written in 2020, was a Windows app. It served its purpose well, but was relegated to Windows. So, someone was asking about this subject - again - on the FB Owners Group and I pointed them to my old app. Then, I got to thinking that a browser extension could be used by nearly everyone with the Chrome browser - or derivatives thereof - regardless of the computer used. Ergo, this extension will work for Windows, Mac, Linux and Chromebook.

Additionally, there are Chrome browser derivatives for Android and iOS that reportedly support extensions - I have not tested this, but for those inclined, try the Orion Browser for iOS and Kiwi Browser for Android. Search the web for details.

NOTE:
“Lots” and “Items” are used interchangeably and refer to each individual item/group of items to be sold. “Images” and “Photos” are used interchangeably and refer to the pictures of items in an aforementioned Lot.

First things first!
Before using this tool, please check Chrome’s download settings, otherwise, you will have to confirm the save location of each image. I assure you, clicking “Save” 100+ times is a real drag.

  • Type chrome://settings/downloads in the URL bar and press enter.
  • Set the option “Ask where to save each file before downloading” to OFF.

How to Use

  1. Log into your CT Seller account
  1. Navigate to your sales page
  2. Choose the dashboard for an auction.
  1. This is the middle icon on the left.
  1. Click the Extensions icon on the Chrome toolbar to open the popup and
    click “CTBids Seller Image Helper”
  1. NOTE: Click the “Pin” icon to put the extension on the toolbar for quick access.

  1. The UI will appear and scan for lots.
  1. If the “More” button is encountered, it will automatically click and continue to scan for lots


  1. Once all lots have been scanned, the UI will present additional controls.

  2. The number of photos to be downloaded is displayed.
  1. If you are not on the seller.ctbids dashboard, an error will be shown.
  1. The auction title will be used for the subfolder name, located in your browser's default download location.
  1. This can be edited BEFORE the download is started.
  1. Check the option to “Skip Pending items” if you only want Lots that are “Ready”
  2. Check the option to “Download all lot images” if you want every photo from each lot.
  1. Be advised, this will download A LOT of photos.
  2. 100 Lots can have over 1000 photos.
  3. Detailed documentation below.
  1. Check the “Create subfolder for each lot” option to save each Lot’s photos to a subfolder named by the Lot’s title. This is very useful with the “Download All...” option above.
  2. Click the “Download Images” button to begin.
  1. All images will be sent to the Browser’s download queue and processed.
  1. A progress bar will display activity while it’s cooking.

  2. When complete, go to your Download folder and you should find a subfolder with the auction title’s name that contains all the cover images with the lot names.
  1. Tip: Control+J will open the Download Manager.


             



This will recursively find and download all photos associated with each lot, adding a numerical suffix to the filename. This allows for the collection of all photos, including the full resolution versions when they’ve been scaled down for thumbnails and bandwidth efficiency. You can easily tell the difference by the filesize, or by hovering your pointer over the image to get details (on Windows. Not sure about Mac)

Or show the Details Pane

This option will create a subfolder for each lot and store the images therein. You will want this when downloading all images, in effort to wrangle the chaos.

BONUS:

Estatesales.NET, ORG and COM all have an option to use the filename as the image description -- they got tired of me pestering them for this feature. .NET is automatic, ORG and COM you have to enable the feature. So, when you advertise your sale on these sites, be sure to enable the option as it makes a much better presentation. Of course, .ORG now has the CT import feature.
